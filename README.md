# Gym Management System

## Project Description
The Gym Management System is a web-based application designed to streamline the management of gym memberships and subscriptions. It provides an intuitive interface for administrators to efficiently manage members, subscriptions, and billing processes.

## Features
- **Membership Management:**
  - Add, edit, and remove gym members.
  - View detailed profiles of gym members.
- **Subscription Management:**
  - Provide details of various gym subscriptions.
  - Manage active and inactive subscriptions.
- **Billing and Receipts:**
  - Generate bills and receipts for payments.
  - Maintain records of all transactions.
- **Information Portal:**
  - Display comprehensive details about the gym and its facilities.

## Technologies Used
- **Frontend:**
  - HTML
  - CSS
  - JavaScript
- **Backend:**
  - PHP
- **Database:**
  - MySQL (phpMyAdmin)
- **Development Environment:**
  - XAMPP

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/gym-management-system.git
   ```
2. Start XAMPP and ensure Apache and MySQL are running.
3. Import the database:
   - Open `phpMyAdmin` and create a new database (e.g., `gym`).
   - Import the provided SQL file (`gym_management.sql`) into the database.
4. Configure the project:
   - Edit the database connection details in the `config.php` file to match your local setup.
5. Deploy the application:
   - Move the project folder to the `htdocs` directory of your XAMPP installation.
6. Access the application:
   - Open a web browser and navigate to `http://localhost/gym-management-system`.

## Usage
1. Log in to the admin dashboard.
2. Use the interface to manage members, subscriptions, and billing.
3. View or edit details as required.
